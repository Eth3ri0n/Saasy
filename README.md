## Tech Stack

**Client:** HTML, CSS, TailwindCSS

**Server:** Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/Eth3ri0n/Saasy.git
```

Go to the project directory

```bash
  cd Saasy
```

Install dependencies

```bash
  npm install
```

Start Tailwindcss

```bash
  npm run start
```
